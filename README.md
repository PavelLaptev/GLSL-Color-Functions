# GLSL-Color-Functions

Color space conversions, metrics, and other utility functions for color in GLSL.

Still very much a work-in-progress!

## Currently Supported
Color Spaces:
* XYZ
* sRGB
* RGB
* L\*a\*b\* (CIELAB)
* LCh (CIELCh)

White Points:
* D50
* D65
